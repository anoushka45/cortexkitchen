# Data Directory

This folder holds project data assets and working directories used by the local CortexKitchen demo setup.

## Layout

- `raw` - source-like input datasets or imported files
- `processed` - transformed outputs generated during data preparation
- `seeds` - seed-oriented artifacts for demo and local environment setup

## Current usage

Most day-to-day demo seeding is driven by the Python scripts in `/scripts`, not by manual edits in this folder. Treat this directory as the home for data assets, not the primary source of application logic.
