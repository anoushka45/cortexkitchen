# Tests

Cross-project test documentation lives here. The active automated backend test suite currently lives in `apps/api/tests`.

## Current reality

- The API has the meaningful automated test coverage today
- The frontend does not yet have a comparable in-repo automated test layer
- This top-level `tests` folder is documentation-oriented right now, not the main executable suite location

## Backend test layout

- `apps/api/tests/unit`
- `apps/api/tests/integration`
